nrm切换npm远利器

发表于 2017-01-11 | 分类于 nrm |

在使用npm时，官方的源下载npm包会比较慢，国内我们基本使用淘宝的源，最近公司内部搭建了一套npm私有仓库。要添加自己公司内部的npm源，公司内部的源不可能把npm官方的npm包都全量同步，故需要npm源之间的切换，如果使用npm registry xxx的话，太不好管理了。nrm是管理npm源切换的利器。使用方法如下：

安装nrm

当然，install 都可以简写为g.

一些常用的命令

  • nrm –help       帮助信息
  • nrm ls         列出现在已经配置好的所有的原地址
  • nrm use        是切换到哪个源上
  • nrm add        添加源
  • nrm del         删除源
  • nrm test        测试源的响应时间，可以作为使用哪个源的参考

第一次下载nrm慢的问题

那么，如果你第一次下载nrm就特别慢，怎么解决呢？
当然了，我们可以手动先配置个速度快的源，默认是npm官方的源，网速不好的话，确实非常慢，所以我们可以配置个淘宝的源（国内还是非常快的）。
下面是手动配置的步骤：

  • 1.打开C:\Users\Administrator文件夹，添加文件.npmrc写入registry=https://registry.npm.taobao.org/(直接创建是不能的，可以用编辑器创建文件)

  • 2.然后就可以了，npm i -g nrm

  • 3.nrm ls 查看npm 的数据源有哪些，标星号的表示正在使用的

  • 4.nrm use 数据源的名字，例如 nrm use taobao

这样就同样安装好了，并且把源切换到了淘宝镜像了。
